Technologies

When I started coding, I was mainly using HTML tables, as many beginners did. A few weeks later I started working with div elements and DOCTYPE 4.01 Transitional. I wasn't aware of alternatives, so I kept working with this method for a long time. Eventually I started using HTML5. At the end of 2017 I was introduced into semantic elements. In 2018 I started learning as hard as I could and that's how I started using HTML & CSS grid views, the CSS preprocessor Sass for better management and usability, gulp.js and Git. I think that I made a lot of progress and am still continuing to learn new technologies.
